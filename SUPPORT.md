# Destek

TrueStretch ile ilgili yardım için önce [README](README.md) içindeki gereksinim ve kullanım bölümlerini kontrol edin.

## Doğru kanal

| Konu | Kanal |
| --- | --- |
| Tekrarlanabilir hata | [Bug report](https://github.com/KayaJR356/TrueStretch/issues/new?template=bug_report.md) |
| Yeni özellik önerisi | [Feature request](https://github.com/KayaJR356/TrueStretch/issues/new?template=feature_request.md) |
| Güvenlik açığı | [SECURITY.md](SECURITY.md) |
| Katkı süreci | [CONTRIBUTING.md](CONTRIBUTING.md) |
| Genel soru | GitHub Issues — uygun şablonu seçin |

## Destek talebine eklenecek bilgiler

- TrueStretch sürümü veya commit SHA
- Windows sürümü ve mimarisi
- GPU ve sürücü sürümü
- Monitör modeli ve bağlantı türü
- Seçilen çözünürlük / yenileme hızı
- Beklenen ve gerçekleşen davranış
- Yeniden üretim adımları
- Hassas bilgi içermeyen ekran görüntüsü veya hata metni

> [!CAUTION]
> Log, ekran görüntüsü veya profil dosyası paylaşmadan önce kullanıcı adı, dosya yolu ve diğer kişisel bilgileri kaldırın.

## Kapsam sınırları

TrueStretch yalnızca Windows ve ekran sürücüsünün sunduğu modlarla çalışır. Özel EDID, firmware değişikliği, donanım arızası veya üçüncü taraf GPU kontrol paneli yapılandırmaları için destek garanti edilmez.
