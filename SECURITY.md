# Güvenlik Politikası

## Desteklenen sürüm

Güvenlik düzeltmeleri en son ana sürüme uygulanır.

## Bildirim

Bir güvenlik açığı bulursanız ayrıntıları herkese açık issue içine yazmayın. GitHub deposundaki **Security → Report a vulnerability** seçeneğini kullanın. Bu özellik etkin değilse, açıklama içermeyen bir issue açarak özel iletişim kanalı isteyin.

## Çalıştırma güvenliği

- EXE henüz kod imzalı değildir.
- İndirdiğiniz dosyanın Release notlarındaki SHA-256 değerini doğrulayın.
- Ekran modu değişikliklerinden önce açık çalışmalarınızı kaydedin.
- Uygulama sadece sürücünün `CDS_TEST` ile kabul ettiği modu uygular.
