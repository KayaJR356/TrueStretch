# Katkı Rehberi

1. Değişikliğiniz için kısa bir issue açın.
2. Depoyu fork edip açıklayıcı bir dal oluşturun.
3. `build.ps1` ile temiz derlemeyi doğrulayın.
4. Ekran modu kodunda değişiklik yaptıysanız desteklenen ve desteklenmeyen birer modu test edin.
5. Pull request açıklamasında kullanıcı etkisini ve yaptığınız kontrolleri belirtin.

Yeni bağımlılık eklemekten kaçının. Win32 yapı alanlarında yapılacak değişiklikler 64-bit Windows'ta doğrulanmalıdır. Ağdan gelen içerik hiçbir zaman çalıştırılmamalı veya HTML olarak doğrudan görüntülenmemelidir.
